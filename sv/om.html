Om

Jag är en systemutvecklare som drivs av att förstå hur saker fungerar på djupet, och att använda den kunskapen för att bygga något som gör nytta på riktigt.

Skicka mejl Ladda ner CV

Bakgrund

Jag har gillat teknik så länge jag kan minnas, men jag gillar också saker så som finans och hälsovård.

Passion för teknik

Teknik har intresserat mig så länge jag kan minnas. Men det var först när jag insåg att kod kan lösa riktiga problem – inom vård, forskning eller finans – som jag verkligen hittade hem.

Viljan att förstå

Jag nöjer mig sällan med att något bara "fungerar". Jag vill veta varför. Det är därför jag gräver ner mig i Linux-konfigurationer och Java bytecode på fritiden.

Värderingar

Principer jag lutar mig mot i mitt arbete.

  • Leverera tydlighet. Kod läses oftare än den skrivs. Jag strävar efter namn och strukturer som gör det enkelt för nästa person att ta över.
  • Samarbete först. Ingen bygger stora system ensam. Jag dokumenterar gärna och delar med mig av kunskap för att hela teamet ska lyckas.
  • Ständig nyfikenhet. Tekniken står aldrig stilla. Jag avsätter tid för att testa nytt, oavsett om det är container-orkestrering eller optimering av AI-modeller.

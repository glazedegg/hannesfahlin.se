Diff-Aware Test Generator

Java · Bytecode-analys · CI/CD

Ett deterministiskt verktyg byggt för CI/CD-pipelines. Den analyserar Java bytecode för att identifiera granulära förändringar i pull-requests och använder SQLite för att kartlägga beroenden. Verktyget genererar automatiskt "skyddsnäts-tester" (t.ex. gränsvärden) specifikt för den ändrade logiken.
